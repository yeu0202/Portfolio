Arduino + Image Processing with Python

This was just the result of some small experimentation with connecting the Arduino to Python. This project used input data from the Arduino to change the strength of some filters in the video input using OpenCV.

Project Video

There is a light sensor attached to the Arduino, and the strength of light detected by the light sensor is sent from the Arduino to Python via the serial port. As seen in the video, the strength of the video filters change depending on the amount of light reaching the light sensor.
