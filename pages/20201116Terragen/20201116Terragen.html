Terragen

This was a class taught by Carl where students experimented with Terragen, a realistic terrain generator.
It uses perlin noise to generate everything. You can then edit the look of the terrain through some built in functions.
I first experimented with creating a nice landscape and making it look nice.
Carl also suggested that we could upload our own images as a terrain, which I did in the last image.

Terrain 1

First image I exported after being somewhat satisfied with the result. The snow, rocks, greenery, sand was all coloured on based on inserted parameters.

Terrain 1, blue sky

Here I colored the sky blue to get a more natural look.

Terrain 1, sunset

I tried to make the sky look like a sunset because I thought making a nice sunset would look nice.

Terrain 1, sunset 2

Here I changed the clouds, this actually took some time as there are a lot of parameters to cloud generation.

Terrain 1, sunset 3

I moved the sun into the field of view and angled it so it would only shine onto some sections of the mountains. I liked the effect the sun gives in this image, you can see the sun rays and the reflection in the water also looks very nice.

Terrain 1, sunset 4

Here I just angled the sun to another position to get a sunset from a different angle.

Source image

Here is the source image I used for generating terrain with a source image. The image has to be in a .tga format, which I converted my image to using photoshop.
(HTML doesn't support tga images so this image is a png)

Image terrain

This is the terrain I made using the image. I basically used the colors of the rainbow to indicate the height of the terrain. This allowed me to actually see some resemblance to the source image. The red bar at the top is a result of the source image and terrain dimensions being different.
At "ground" level, the terrain is extremely rough and has lots of towers and pits due to the noise in the image.

Additional comments

This exercise was an interesting insight into how terrain can be generated. The terrain generated using perlin noise looked very realistic. It was also pretty fun to see how an image can look like as terrain.
