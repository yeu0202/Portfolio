Image Processing with p5.js

This was part of the Small Object of Delight project I did for Magnus's class. I used p5.js to create a series of camera filters.
The code is probably terribly inefficient because I wrote the filters myself, and I used a weird way to store the functions (putting them in an array).
Github link: https://yeu0202.github.io/Processing_0/ImageProcessing/

Additional comments

In the end, this method of creating and messing around with camera filters was discarded because the frame rate was horrendous, especially for some more intensive filters. Nonetheless, this part of the project was fun as I learned how some filters work on the pixel level.
