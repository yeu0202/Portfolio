Fractal Exploration Project

This is the Fractal Exploration Project, the Final Project for Magnus's module. I worked on this project with Masha and Paulina.
I chose to work on fractals as they fascinated me, and I thought it would be cool if there was a game that could allow the player to just explore them freely. The three of us created our own fractals separately due to not being in the same place. Then, we combined them into one single application in Unity.
I worked on creating a colorful cubic display based off of the concept of Koch fractals. The object would pulse to the sound of the music playing in the background. I also created the player object, which was made of two nested icosahedrons. The icosahedrons would also pulse to the music. I combined everyone's fractals together into one Unity game, then added some instructions and features like settings and customization to the game.

I learned quite a bit about different tools in Unity like the skybox, camera, and the line renderer. For me it was quite a technical project, figuring out how to manipulate and enhance code from a tutorial to suit my own needs.

Project Video

This is a video that gives an overview of our process and final result of our project.

I enjoyed creating the icosahedron, which is actually made up of 20 triangles. I created the icosahedron by using the dihedral angle of an icosahedron, which is the angle between two faces. With specific rotations about the center of the object, the 20 triangles would form an icosahedron.
I made some snowflakes by applying the Koch fractal generator script onto a hexagon multiple times. This process allowed me to create a series of different snowflakes by changing a few small values. I also enjoyed making the player controller. I followed a simple tutorial by Brackeys and adjusted some settings to create a nice third person controller for our player object. Instead of moving the ball normally, I also changed the code so there is a force applied on the ball from a direction, which would make it roll better.
I learned about changing the skybox image in this project, for our skybox image I used a free asset available on the asset store.
I also enjoyed receiving feedback from my friends after showing them this project, and adding features based on their responses.

Future Development

We could definitely develop this project further. The game doesn't feel fully complete yet, and could also use some fine tuning on some of the fractal objects. The game also takes quite a lot of processing power, and it would be nice to be able to optimise it better. Unfortunately, some of the objects imported from other software created by my groupmates didn't end up as nice as they could have, as some shaders and scripts could not be exported from Blender.

Additional comments

I learned a lot about Unity packages and about Unity itself from working on this project. The cinemachine package was really helpful in making the third person camera, but was also really helpful in implementing some features like the pause camera, changing fov and sensitivity. I didn't have many technical difficulties, and most issues I had were able to be solved quickly through some searches on online forums.
I definitely will develop this fractal idea in the future, as I don't think I have explored the full potential of beautiful fractals.
