Fast Fourier Transform (FFT) Test

This was a small test file I made while working on another project (Peter's assessment 1). I decided to give it a separate page.
This file was made for me to test what variables I could change and ways I could visualise sound.

Project Video

I learned how to apply the FFT on audio from a tutorial by Peer Play.
It basically turns the audio from amplitude/time spectrum into the amplitude+frequency/time spectrum. This visualises the audio in a nice way that is pretty easy to understand.
Then I looked up ways to manipulate different visual aspects of objects in Unity and applied them to the visualization objects.
