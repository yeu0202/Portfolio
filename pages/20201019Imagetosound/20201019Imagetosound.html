Image to Sound

This was a class taught by Carl where students experimented with converting images to sound files and back.
A filter or effect was applied on the sound file before converting it back to an image.
Software used was Adobe Photoshop and Adobe Audition.

Original Image

Picture I took at a squash court in the NUS.

Grayscale Image

Image in grayscale, as it was easier to work with sound in grayscale.

Echo effect

You can see elements of the image 'echoed' horizontally.

Invert and Minus 10db

The sound was inverted, which inverted the colors in the image.
The minus 10db effect removed some detail as it probably shifted all the values closer to one end.

Flanger

The flanger effect in sound mixes the same signal with itself but one is slightly delayed with a changing period.
These can be noticed in the triangles behind some lines in the image.
I think that the height of a triangle might have something to do with the way sound is sampled.

Phaser

The phaser filters the signal by creating some peaks and troughs in the frequency spectrum.
This image mostly looks like the original grayscale image but there is some distortion which is more noticable at harder edges.

Phaser Zed Leppelin

The phaser filters the signal by creating some peaks and troughs in the frequency spectrum.
This phaser filter is a lot stronger than the default one.
There seems to be some sort of echo effect built into the phaser effect.

Low Pass Filter

I removed high frequency sounds using a low pass filter.
I expected the image to come out with some thin lines removed/blurred.
While this has happened on the person and the edges of the court glass boundaries, there are some grainy artefacts on the walls and the floor that I didn't expect.

Low Pass Filter 2

This image was processed through a stronger low pass filter.
Both the blurring that was expected and the artefacts are stronger.
I find this effect pretty interesting, further investigation will be needed to understand why the artefacts happen.

Additional comments

There are plenty of other audio filters out there to test. It would be nice if there was an easy way to process colour images that makes sense. I'm sure that can be done with a bit of programming to process the images into audio files and back in the way I want.
It would also be interesting to see if you could turn audio data into images and put an image filter over them, then convert them back. One issue with that is that it might be harder to detect the subtle changes in sound.
