Whatsapp Text Analysis

This was a small project I did because I thought it would be interesting to be able to look at some of the data of my text history with some of my friends.
I created a small script that analysed the text file you get from downloading a whatsapp text.

Project Screenshot

The script goes through the text file and stores each word in a dictionary and stores a counter for each word. At the end, the program outputs a simple analysis of the most used words and the total word count for each person.

Additional comments

This was a pretty fun, quick, and interesting exercise in programming that also allowed me to gain a little insight into some of me and my friends' messaging habits. I uploaded this Python program onto github.
